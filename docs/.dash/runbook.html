Weekly Run

Use these commands to refresh feeds, lineup artifacts, and the public dashboard. The flow is idempotent; rerun as needed.

make verify-all WEEK=auto
make weekly-all WEEK=auto
make pages-open

After weekly-all completes, Pages links should reflect the latest manifest, lineup, waivers, trades, and metrics.

Gameday (T-120)

Kick off the late-swap watcher and publish updates roughly two hours before the first kickoff.

make gameday WEEK=auto WIN=120

For a quick status on Pages links once the run finishes:

make pages-open

Troubleshooting

If Yahoo roster/FA sync is missing, ensure credentials are present and rerun:

make yahoo-setup
make yahoo-auth
make yahoo-sync WEEK=auto

Whenever weekly-all fails early, inspect the guard log printed in the console (under logs/assistant/) and rerun the individual target noted in the error.

Freshness Guardrails

Freshness checks warn when critical feeds are stale, empty, or for the wrong week. Run them anytime before publishing:

make freshness WEEK=auto

To hard-fail on critical freshness misses (optional feeds like Vegas/Weather stay advisory), enable the enforcement flag:

FF_FRESHNESS_ENFORCE=1 make verify-all
